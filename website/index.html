// A simple HTTP server
fn main() {
    let fd = http_bind(8080)
    print("listening on :8080")

    while 1 == 1 {
        let c = http_accept(fd)
        let _ = http_respond_json(c, 200, "{\"ok\":true}")
        let _ = http_close(c)
    }
}

What Mako gives you

Mako is a young language. Here is what works today.

⚡

Compiles to C

Mako source compiles to C, then to a native binary via clang. Incremental compilation is on by default.

🔒

Memory safety without GC

Ownership, arenas, and reference counting. No garbage collector. Memory is freed deterministically.

⚗

Structured concurrency

crew blocks ensure all spawned tasks complete before the block exits. Channels and actors are built in.

🌐

Networking in the stdlib

TCP, UDP, HTTP/1.1, TLS, and WebSocket are included. No external frameworks required for basic servers.

📊

Database support

Drivers for SQLite and PostgreSQL are included. Redis support is available through the RESP protocol.

🚀

Cross-compilation

Target macOS, Linux, and WebAssembly from a single machine. Windows support is in progress.

Meet the Mako

Named after the Jawed Limia (Limia mandibularis), a small freshwater fish endemic to Hispaniola. Like its namesake, Mako is small and does well in its environment.

Try Mako

Requires Rust (for the compiler) and clang.

$ git clone https://github.com/loreste/mako && cd mako && make install

Then: mako init myapp && mako run myapp/main.mko
